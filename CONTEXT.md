# TTSMM-EX

TTSMM-EX is a desktop utility for configuring TerraTech mods, validating collections, and launching the game with a predictable setup.

## Language

**View stage fill contract**:
The rule that every main workspace view occupies the full application content area.
_Avoid_: partial route surface, floating route shell

## Relationships

- A **View stage fill contract** applies to Settings, Block Lookup, Collection, and startup loading views.

## Example dialogue

> **Dev:** "Should Settings size itself to its form content?"
> **Domain expert:** "No, Settings is a main workspace view, so the **View stage fill contract** applies."

## Flagged ambiguities

- "It only uses part of the window" was resolved as a violation of the **View stage fill contract**, not a window-size or data-loading issue.
